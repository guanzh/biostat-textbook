21.1 相邻观测为何共享信息

第 18 章处理了层级结构（植株嵌套于小区）。但保护数据中还有另一种依赖：时间和空间上的邻近。

  • 同一相机样点今年的检出率和去年的检出率相关：时间自相关
  • 相邻两个相机样点的检出率比远距离样点更相似：空间自相关
  • 同一景观内修复地和对照地的变化趋势相关：空间共享

忽视时空依赖的后果与忽视层级结构相同：标准误过小，置信区间过窄，P 值过小。你以为你有 1080 个独立观测：实际上相邻观测共享信息，有效样本量远小于 1080。

camera <- read.csv("../data/teaching/camera_monitoring_long.csv",
                   stringsAsFactors = FALSE)
rest <- read.csv("../data/teaching/restoration_site_period.csv",
                 stringsAsFactors = FALSE)
识别依赖时间与空间
选择尺度趋势/季节/梯度
调整模型相关结构与平滑
分块验证空间留出
限制外推地图与边界

学完本章后，你能够：

  1. 识别时间自相关和空间自相关，并解释它们如何缩小有效样本量。
  2. 区分趋势、季节和随机波动：不同时间尺度需要不同的模型。
  3. 理解为什么随机拆分训练/测试集会泄漏空间信息。
  4. 设计按年份或空间区块留出的验证方案。
  5. 识别外推的边界：地图空白区不等于低风险。

21.2 趋势、季节与时间自相关

时间序列的三个成分：

趋势（trend）：长期的系统性变化：“三年间检出率是否在下降？”

季节（seasonality）：周期性波动：“每年第三 occasion（雨季）检出率是否系统性偏低？”

自相关（autocorrelation）：相邻时间点的残差相关：“今年的高检出率增加了明年也高的概率。”

如果忽视自相关，趋势检验的标准误会偏小：你以为是独立的三年的独立证据，实际上它们共享信息。

# 按年份聚合检出率
yearly_detection <- aggregate(
  detected ~ year + station_id,
  data = camera,
  FUN = mean
)

# 画年度轨迹
year_means <- aggregate(detected ~ year, data = yearly_detection, FUN = mean)
plot(year_means$year, year_means$detected, type = "b",
     col = "#2e7d32", lwd = 2, pch = 16,
     xlab = "年份", ylab = "平均检出率",
     main = "年度检出率趋势",
     ylim = c(0, 1))

# 检查相邻年份的相关
y1 <- yearly_detection$detected[yearly_detection$year == 1]
y2 <- yearly_detection$detected[yearly_detection$year == 2]
cat("年份 1-2 相关:", round(cor(y1, y2), 3), "\n")

21.3 空间梯度、距离与空间自相关

空间自相关：距离越近的样点越相似。原因可能是： - 共享生境条件（土壤、气候） - 物种扩散（相邻样点可能有同一个体） - 未测量的空间梯度

空间自相关的诊断：残差的半变异函数（variogram）或 Moran’s I。如果近距离残差比远距离残差更相似，说明存在空间自相关。

# 按样点聚合坐标和检出率
station_info <- unique(camera[, c("station_id", "x_km", "y_km")])
station_detection <- aggregate(
  detected ~ station_id,
  data = camera,
  FUN = mean
)
station_data <- merge(station_info, station_detection, by = "station_id")

# 计算样点间距离矩阵（简化版：取前 10 个样点演示）
n_demo <- 10
demo_stations <- station_data[1:n_demo, ]
dist_matrix <- as.matrix(dist(demo_stations[, c("x_km", "y_km")]))

# 检查距离与检出率差异的关系
dist_vec <- dist_matrix[lower.tri(dist_matrix)]
detect_diff <- abs(outer(demo_stations$detected, demo_stations$detected, "-"))
detect_diff_vec <- detect_diff[lower.tri(detect_diff)]

plot(dist_vec, detect_diff_vec,
     xlab = "样点间距离 (km)", ylab = "检出率差异",
     pch = 16, col = "#4d8294")
cat("距离与检出率差异的相关:",
    round(cor(dist_vec, detect_diff_vec), 3), "\n")

21.4 采样尺度决定可见过程

尺度（scale） 是时空分析中最关键的概念：

  • 采样尺度（grain）：单个观测的分辨率：相机样点覆盖约 0.01 km²
  • 研究范围（extent）：所有样点覆盖的总面积：约 30 × 30 km
  • 尺度错配：用 1 m² 样方研究景观尺度的过程，或用 30 km 网格研究微生境选择

不同过程在不同尺度上显现： - 微生境选择 → 样方尺度（米） - 种群动态 → 样点尺度（百米到千米） - 分布变化 → 景观尺度（千米到数十千米）

21.5 时空依赖怎样进入模型

处理时空依赖的常见方法（本章只要求识别，不要求实现）：

  • 固定效应：把时间或空间坐标作为协变量：“检出率随 x 坐标线性变化”
  • 随机效应：样点或年份作为随机截距：第 18 章的方法
  • 相关结构：残差的自回归（AR1）或空间相关结构
  • 平滑项：用 GAM（广义可加模型）拟合非线性时空趋势
  • 随机场：用高斯随机场建模连续空间变化

21.6 分块验证与地图不确定性

随机拆分训练/测试集是时空数据中最常见的验证错误。

如果 1080 行数据中，同一相机样点的 18 条记录被随机分到训练集和测试集。模型在测试集上表现”很好”，因为它已经”见过”这个样点的其他记录。这不是在验证模型的预测能力。这是在验证数据录入的一致性。

正确的验证方案： - 按年份留出：用第 1-2 年训练，预测第 3 年 - 按空间区块留出：用部分景观训练，预测留出的景观 - 按样点留出：用部分样点训练，预测留出的样点

# 演示：随机拆分 vs 空间留出
set.seed(20260620)

# 错误：随机拆分 1080 行
idx_random <- sample(1:nrow(camera), size = 0.7 * nrow(camera))
cat("随机拆分：训练集", length(idx_random), "行，测试集",
    nrow(camera) - length(idx_random), "行\n")
cat("问题。同一样点的不同 occasion 可能分散在训练和测试集中。\n\n")

# 正确：按样点留出
all_stations <- unique(camera$station_id)
train_stations <- sample(all_stations, size = 0.7 * length(all_stations))
test_stations <- setdiff(all_stations, train_stations)
cat("空间留出：训练集", length(train_stations), "个样点，测试集",
    length(test_stations), "个样点\n")
cat("同一样点的所有记录完整地属于训练集或测试集。\n")

21.7 外推和监测更新

时空模型的外推边界：

  • 时间外推：用 3 年数据预测第 4 年：假设趋势延续。如果发生了异常气候事件或管理变化，预测可能严重偏差。
  • 空间外推：用 60 个样点预测未布点区域：假设空间关系在预测范围内一致。如果未布点区域有不同生境，预测不可靠。
  • 地图空白区 ≠ 低风险，没有样点的区域，模型的不确定性最大：不要因为地图上没有”热点”就认为那里安全。

观测制度变化可能伪装生态趋势。如果第 3 年换了相机型号、改了调查时间、或换了调查员：检出率的变化可能是观测制度变化，不是生态变化。

本章小结

  • 相邻观测共享信息。时间自相关和空间自相关缩小有效样本量：忽视它们等于制造伪重复。
  • 不同过程在不同尺度上显现。采样尺度和研究范围决定了你能看到什么过程。
  • 随机拆分训练/测试集泄漏空间信息。正确的验证按年份或空间区块留出。
  • 外推必须明确标注边界。时间外推假设趋势延续；空间外推假设关系一致。
  • 观测制度变化可能伪装生态趋势。比较年份之前，先检查调查方法是否一致。

练习

  1. 识别层。以下数据各存在什么类型的依赖？

    1. 同一相机样点 3 年的检出率
    2. 相邻 5 个样方的物种丰富度
    3. 同一植株 3 个时间点的株高

  2. 解释层。为什么随机拆分 1080 行会高估模型的预测能力？

  3. 应用层。为相机数据设计一个按年份留出的验证方案。说明训练集和测试集各包含什么。

  4. 诊断层。残差的 Moran’s I 显著为正。这意味着什么？你应该怎么处理？

  5. 综合层。为修复地项目设计一个多年空间监测方案。包含：样点布局、时间频率、验证策略、以及外推边界。
