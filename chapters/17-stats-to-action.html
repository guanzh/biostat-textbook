黑豹提醒：P = 0.03，因此立即全区推广综合防治——这个推理跳过了效应大小、成本、副作用和可逆性。

17.1 统计结论不是行动命令

前面 16 章建立了从设计到数据分析的完整链条。但链条的最后一环——“这些证据意味着什么行动”——不是统计软件能回答的。

统计结论： “综合防治使虫口密度降低了 X 头/株（95% CI [下限, 上限]，P = 0.003）。”

行动决策还需要： - 这个效应在生态上是否重要？（X 头/株的降低是否足以阻止树木死亡？） - 成本是否可接受？（每公顷 Y 元——预算够吗？） - 副作用是否可接受？（非目标生物减少了 Z%——这会影响天敌种群吗？） - 效果是否可逆？（如果推广后发现问题，能停下来吗？） - 是否有更好的替代方案？（生物防治单独使用效果稍差但成本低很多？）

本章不设置最终考核。 四个贯穿项目各自承担一种决策难点，帮助你练习”从证据到行动”的完整推理。

提取效应大小与区间
对照阈值最小有意义效应
权衡代价成本/风险/伦理
选择行动行动/监测/停止
安排更新适应性管理

17.2 效应大小与生态意义

P 值回答”效应是否可能为零”——它不回答”效应是否重要”。

最小有意义效应（minimum meaningful effect）： 在管理上有意义的最小变化。例如： - 虫口密度降低 < 1 头/株 → 管理上无意义（树木仍然受害） - 虫口密度降低 ≥ 3 头/株 → 管理上有意义（树木受害率明显下降）

判断流程： 1. 估计效应和置信区间 2. 将区间与最小有意义效应比较 3. 如果整个区间都在有意义范围内 → 证据支持重要效果 4. 如果区间包含零或包含无意义范围 → 证据不足 5. 如果区间完全在无意义范围内 → 证据支持效果不重要

17.3 区间和行动阈值

行动阈值（action threshold）将连续的证据转化为二元的行动决策：

区间相对于阈值的位置 含义 行动
区间完全高于阈值 效果很可能大于阈值 考虑行动
区间跨越阈值 效果可能大于也可能小于阈值 证据不足，继续监测
区间完全低于阈值 效果很可能小于阈值 不行动或换方案

注意： 阈值是管理决策，不是统计输出。统计提供效应和区间——管理者设定阈值。

17.4 成本、风险、伦理与可逆性

四个贯穿项目的决策难点：

黄黄草（外推风险）： 苗圃试验显示 40% 光照开花最多。能否建议野外搭建遮阴网？ - 苗圃 → 野外外推有损耗（土壤、竞争、物候不同） - 遮阴网的成本和生态影响？ - 建议：先选 2–3 个野外样地做小规模验证

红外相机（探测不确定性）： 检出率下降——是种群下降还是探测概率下降？ - 不能区分 → 需要改进监测（增加重复调查、校准相机） - 在区分之前，不能宣布”种群下降”

森林防治（多指标冲突）： 综合防治虫口最低，但成本最高，非目标影响中等。 - 生物防治效果稍差但成本低、副作用小——可能是更好的选择 - 取决于预算和生态优先级

修复地（景观异质性）： BACI 效果在景观间差异大——景观 3 几乎无效果。 - 不能假设”修复在所有地方都有效” - 需要分析景观 3 为什么不同（土壤？水文？初始条件？）

17.5 多指标冲突

保护决策中经常出现指标冲突： - 虫口密度降低 vs 非目标生物减少 → 哪个优先？ - 丰富度增加 vs 本地种比例不变 → 修复成功了吗？ - 成本低 vs 效果强 → 预算有限时选哪个？

不能把多指标合成一个总分。 权重反映价值判断——统计不能替你决定”天敌保护比虫口降低重要 1.5 倍”。

17.6 证据不足时的下一步

三种证据状态和对应的行动：

  1. 证据支持重要效果： 区间完全在阈值之上 → 考虑行动（同时安排监测）
  2. 证据支持效果很小或不存在： 区间完全在阈值之下 → 不行动或换方案
  3. 证据不足（区间太宽）： 区间跨越阈值 → 继续监测，增加样本量或改进设计

适应性管理（adaptive management）： 把行动本身当作试验——行动 + 监测 + 根据新证据调整。不要求”一次做对”，要求”做的时候能学到东西”。

17.7 一页证据简报

将统计结果转化为管理者能看懂的简报：

栏目 内容
管理问题 一句话：我们要决定什么？
设计与数据 什么设计、多少独立单位、什么时间范围
主要效应 效应大小 + 95% CI + 单位
区间与阈值 区间相对于行动阈值的位置
诊断与局限 模型假设、异常值、未控制的混杂
成本和副作用 经济成本、非目标影响、外推风险
建议行动 行动 / 继续监测 / 停止——附理由
重新评估条件 什么新证据会改变这个建议？

本章小结

  • 统计结论不是行动命令。 P 值和效应量是决策的输入——成本、风险、伦理和可逆性共同决定行动。
  • 最小有意义效应是管理概念，不是统计概念。 统计提供效应和区间——管理者设定”多大才算大”。
  • 区间相对于阈值的位置决定行动。 完全高于→行动；跨越→继续监测；完全低于→不行动。
  • 多指标冲突时不能合成总分。 权重反映价值判断——透明地列出各指标的得失。
  • 证据不足是合法结论。 “我们目前不知道”比”我们假装知道”更有助于保护决策。
  • 适应性管理：行动 + 监测 + 调整。 不要求一次做对——要求做的时候能学到东西。

练习

  1. 识别层。 以下结论各属于哪种证据状态（支持重要效果/支持效果很小/证据不足）？

    1. 效应 = 5，95% CI [3, 7]，阈值 = 2
    2. 效应 = 1，95% CI [−2, 4]，阈值 = 5
    3. 效应 = 0.5，95% CI [0.1, 0.9]，阈值 = 3

  2. 解释层。 “P = 0.03，因此立即全区推广综合防治。” 这个推理跳过了哪些步骤？

  3. 应用层。 为修复地 BACI 效果写一份一页证据简报。包含所有 8 个栏目。

  4. 诊断层。 综合防治虫口最低但成本最高，生物防治效果中等但成本低。管理者说”那就选综合防治”。请列出这个决策中隐含的价值判断。

  5. 综合层。 为黄黄草苗圃试验的结果写一份行动建议。明确区分”苗圃内的结论”和”野外推广的建议”——外推需要什么额外证据？
