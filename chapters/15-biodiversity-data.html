15.1 “多样性”是哪种变化

“生物多样性”可能是保护生物学中最常用也最模糊的词。它可以指：

  • 物种丰富度（species richness），有多少种：最简单的指标
  • 均匀度（evenness）：各物种的个体数是否均匀分布
  • 组成（composition）：具体是哪些物种：两个样方可能丰富度相同但物种完全不同
  • 功能多样性（functional diversity）：物种在生态系统中的角色是否多样

在开始计算任何指数之前，先明确：你要回答的保护问题是什么？

  • “修复是否增加了物种数？” → 丰富度
  • “修复是否改变了优势种？” → 均匀度 + 组成
  • “修复地群落是否变得更像参照群落？” → 组成相似度
rest <- read.csv("../data/teaching/restoration_site_period.csv",
                 stringsAsFactors = FALSE)
comm <- read.csv("../data/teaching/restoration_community_long.csv",
                 stringsAsFactors = FALSE)
定义问题哪种多样性
统一努力积累与稀释
计算指标丰富度/Shannon/均匀度
比较组成距离与排序
限制行动指数≠修复成功

15.2 丰富度与努力量

丰富度是最直观的多样性指标，但它对调查努力量极度敏感。调查 20 个样方发现的物种数必然多于调查 5 个样方。这不意味着多样性更高，只意味着你花了更多努力。

物种积累曲线（species accumulation curve）：随着样方数增加，累计发现的物种数如何增长。如果曲线趋于平缓，说明努力量基本足够；如果仍在快速上升，说明还有大量物种未被发现。

稀释法（rarefaction）：将不同样本的丰富度标准化到相同的个体数或样方数，使它们可比。

# 从群落长表重建群落矩阵
comm_present <- comm[comm$abundance > 0, ]

# 每个样方的丰富度
comm_richness <- aggregate(
  species_id ~ landscape_id + site_id + period + quadrat_id,
  data = comm_present,
  FUN = function(x) length(unique(x))
)
names(comm_richness)[5] <- "richness_from_comm"

# 与 site_period 数据核对
head(comm_richness)

# 修复后修复地 vs 对照地的丰富度
comm_richness$status <- substr(comm_richness$site_id,
                                nchar(comm_richness$site_id),
                                nchar(comm_richness$site_id))
comm_richness$status <- ifelse(comm_richness$status == "R", "restored", "control")
aggregate(richness_from_comm ~ period + status,
          data = comm_richness, FUN = mean)

15.3 Shannon 与均匀度

Shannon 指数（H’）：H’ = −Σ pᵢ ln(pᵢ)，其中 pᵢ 是物种 i 的相对丰度。H’ 同时反映丰富度和均匀度：物种越多、分布越均匀，H’ 越高。

Pielou 均匀度（J’）：J’ = H’ / ln(S)，其中 S 是丰富度。J’ 接近 1 表示各物种丰度相近，接近 0 表示少数物种占绝对优势。

关键局限：相同的 Shannon 指数可以来自完全不同的群落：10 种各 10 个个体，和 5 种各 20 个个体 + 5 种各 0 个（不可能），H’ 完全不同。但两个不同的群落也可能碰巧有相同的 H’。

# 计算一个样方的 Shannon 指数（示例）
# 取第一个样方的数据
sample_quadrat <- comm[
  comm$landscape_id == 1 &
  comm$site_id == "L1_R" &
  comm$period == "after" &
  comm$quadrat_id == 1, ]

# Shannon 指数
p <- sample_quadrat$abundance / sum(sample_quadrat$abundance)
p <- p[p > 0]  # 排除零
H <- -sum(p * log(p))
S <- length(p)
J <- H / log(S)

cat("丰富度 S:", S, "\n")
cat("Shannon H':", round(H, 3), "\n")
cat("Pielou J':", round(J, 3), "\n")

15.4 α 与 β

  • α 多样性：单个样方或单个地点内的多样性：“这个样方有多少种？”
  • β 多样性：样方之间的物种组成变化：“修复地和对照地的物种组成有多不同？”
  • γ 多样性：整个景观或区域的总多样性：“整个保护区有多少种？”

α 和 β 回答不同尺度的问题。修复可能提高了 α（每个样方种类增加），但如果所有样方都变成相同的物种，β 反而下降了。

15.5 物种积累与稀释

# 简单物种积累曲线（固定随机顺序）
set.seed(20260620)
# 取修复后修复地的样方
comm_ra <- comm[comm$period == "after" & grepl("_R$", comm$site_id), ]
quadrats_ra <- unique(comm_ra[, c("landscape_id", "site_id", "quadrat_id")])

# 随机排列样方顺序
n_quadrats <- nrow(quadrats_ra)
order_random <- sample(1:n_quadrats)
cumulative_richness <- numeric(n_quadrats)
species_seen <- character(0)

for (i in 1:n_quadrats) {
  idx <- order_random[i]
  q <- quadrats_ra[idx, ]
  spp <- comm_ra$species_id[
    comm_ra$landscape_id == q$landscape_id &
    comm_ra$site_id == q$site_id &
    comm_ra$quadrat_id == q$quadrat_id &
    comm_ra$abundance > 0
  ]
  species_seen <- union(species_seen, spp)
  cumulative_richness[i] <- length(species_seen)
}

plot(1:n_quadrats, cumulative_richness, type = "l",
     col = "#2e7d32", lwd = 2,
     xlab = "样方数", ylab = "累计物种数",
     main = "修复后修复地：物种积累曲线")

15.6 群落矩阵、距离、排序与置换

群落矩阵：行 = 样方，列 = 物种，值 = 丰度。这是群落分析的标准数据格式。

距离矩阵：每对样方之间的相异度。Jaccard 距离基于存在/不存在；Bray-Curtis 距离基于丰度。选择哪个距离取决于你关心”物种是否出现”还是”丰度是否变化”。

排序（ordination）：将高维群落数据降维到 2-3 维，可视化样方之间的相似关系。排序只展示结构：不检验假设。

置换检验：在群落分析中，置换必须尊重设计结构，不能随机交换不同景观或不同时期的样方。置换单位是样方，限制在 BACI 设计允许的交换范围内。

本章小结

  • “多样性”必须被拆解。丰富度、均匀度、组成、功能：不同指标回答不同问题。
  • 丰富度对努力量敏感。调查 20 个样方 vs 5 个样方：差异可能来自努力量而非生态。
  • 同一指数可以来自不同群落。Shannon 更高 ≠ 修复成功：必须同时看组成。
  • 排序只展示结构，不检验假设。置换检验必须尊重设计结构（景观、BACI）。
  • 不超过三个互补指标。丰富度 + 本地种比例 + 一个组成指标：就够了。

练习

  1. 识别层。以下问题各适合什么多样性指标？

    1. “修复地是否比对照地有更多物种？”
    2. “修复地群落是否变得更像天然参照群落？”
    3. “修复是否减少了优势种的占比？”

  2. 解释层。为什么”修复地 20 个样方、对照 5 个样方，直接比较总物种数”是错的？

  3. 应用层。用群落长表计算修复后修复地和对照地的 Shannon 指数均值。它们有差异吗？这个差异能说明什么、不能说明什么？

  4. 诊断层。两个样方的 Shannon 指数都是 2.0。它们的群落一定相似吗？为什么？

  5. 综合层。为修复项目选择不超过三个互补的多样性指标。说明每个指标回答什么问题，以及它们的局限性。
