第 20 章 贝叶斯统计与模型诊断

旧经验与新数据冲突 → 概率更新与检查 → 有敏感性的决策概率
① 写出先验
根据已有知识设定先验分布——弱信息、实质先验、或均匀
→
② 接入数据
似然函数 × 先验 → 后验——数据更新了我们的信念
→
③ 比较行动
P(效果 > 阈值 | 数据)——直接回答管理者的问题
黑豹提醒：贝叶斯不自动解决小样本问题——弱数据 + 强先验 = 后验就是先验。

20.1 概率更新服务哪个决策

前面 19 章使用的是频率学派（frequentist）框架：参数是固定的未知数，数据是随机的，P 值是”如果零假设为真，观察到当前数据的概率”。

贝叶斯框架翻转了这个逻辑：参数是随机的（表达我们的不确定性），数据是固定的（已经观察到的）。 我们从先验信念出发，用数据更新信念，得到后验信念。

贝叶斯框架直接回答管理者真正关心的问题： - “综合防治效果超过阈值的概率是多少？” → P(效果 > 阈值 | 数据) - “下一个样方的丰富度会落在什么范围？” → 后验预测分布

频率学派不能直接回答这些问题——它只能回答”如果零假设为真，数据的概率”。

forest <- read.csv("../data/teaching/forest_pest_trial_long.csv",
                   stringsAsFactors = FALSE)
写出先验可解释可检查
接入数据似然×先验→后验
检查预测先验预测与后验预测
诊断计算R-hat 与有效样本量
比较行动决策概率与敏感性

学完本章后，你能够：

  1. 解释先验、似然和后验的概念，以及贝叶斯更新的直观含义。
  2. 区分弱信息先验、实质先验和均匀先验，并说明先验必须可解释和可检查。
  3. 解释可信区间与频率学派置信区间的区别。
  4. 理解后验预测检查的目的——模型生成的数据是否像真实数据。
  5. 识别 R-hat 和有效样本量在 MCMC 诊断中的作用。

20.2 先验、似然与后验

贝叶斯定理：P(参数 | 数据) ∝ P(数据 | 参数) × P(参数)

  • 先验 P(参数)： 在看到数据之前，我们对参数的信念
  • 似然 P(数据 | 参数)： 给定参数，观察到当前数据的概率——与频率学派相同
  • 后验 P(参数 | 数据)： 在看到数据之后，我们对参数的更新信念

共轭先验： 当先验和后验属于同一分布族时，后验有解析解。Beta-Binomial 是最经典的例子——适合演示贝叶斯更新的核心逻辑。

20.3 先验必须可解释和可检查

三种先验类型：

弱信息先验（weakly informative）： 提供合理的范围约束，但不强烈影响结果。例如 Normal(0, 10)——“效应不太可能超过 ±20”。

实质先验（substantive prior）： 基于已有研究或专家知识。例如”根据 5 个已发表的苗圃试验，光照效应约为 −0.5 ± 0.3”。

均匀先验（flat prior）： 对所有可能值赋予相等的权重。看似”客观”，但实际隐含了”大效应和小效应同样可能”的假设——这在生态学中很少合理。

先验预测检查： 在见到数据之前，从先验生成模拟数据——这些模拟数据是否在合理范围内？如果先验预测的虫口密度经常为负数，说明先验不合理。

# Beta-Binomial 共轭示例：综合防治的"成功"概率
# 定义"成功"：小区受害率 < 30%
f3 <- forest[forest$visit == 3, ]
plot_damage <- aggregate(
  damaged ~ block_id + plot_id + treatment,
  data = f3,
  FUN = function(x) c(n = length(x), damaged = sum(x >= 3))
)
plot_damage$n <- plot_damage$damaged[, "n"]
plot_damage$damaged_count <- plot_damage$damaged[, "damaged"]
plot_damage$damaged <- NULL

# 综合防治的小区
int_plots <- plot_damage[plot_damage$treatment == "integrated", ]
# 成功 = 受害树 ≤ 6 株（即受害率 ≤ 30%）
int_plots$success <- int_plots$damaged_count <= 6

# 三种先验
# 先验 1：弱信息——Beta(1, 1) 即均匀分布
# 先验 2：乐观——Beta(2, 1)，倾向于高成功率
# 先验 3：悲观——Beta(1, 2)，倾向于低成功率

# 数据：n 个小区中 k 个成功
k <- sum(int_plots$success)
n <- nrow(int_plots)

cat("数据：", n, "个综合防治小区中", k, "个成功\n\n")

# 后验：Beta(prior_a + k, prior_b + n - k)
priors <- list(
  "弱信息 Beta(1,1)" = c(1, 1),
  "乐观 Beta(2,1)" = c(2, 1),
  "悲观 Beta(1,2)" = c(1, 2)
)

par(mfrow = c(1, 3))
for (name in names(priors)) {
  ab <- priors[[name]]
  post_a <- ab[1] + k
  post_b <- ab[2] + n - k

  x <- seq(0, 1, length.out = 200)
  # 先验
  prior_dens <- dbeta(x, ab[1], ab[2])
  # 后验
  post_dens <- dbeta(x, post_a, post_b)

  plot(x, post_dens, type = "l", col = "#2e7d32", lwd = 2,
       xlab = "成功率", ylab = "密度",
       main = name, ylim = c(0, max(post_dens, prior_dens) * 1.2))
  lines(x, prior_dens, col = "gray60", lty = 2)
  legend("topright",
         legend = c("先验", "后验"),
         col = c("gray60", "#2e7d32"),
         lty = c(2, 1), lwd = c(1, 2), cex = 0.8)

  # 后验均值
  post_mean <- post_a / (post_a + post_b)
  abline(v = post_mean, col = "#c86652", lty = 3)
}

cat("\n后验成功率 > 0.5 的概率：\n")
for (name in names(priors)) {
  ab <- priors[[name]]
  post_a <- ab[1] + k
  post_b <- ab[2] + n - k
  prob <- 1 - pbeta(0.5, post_a, post_b)
  cat(name, ":", round(prob, 3), "\n")
}

20.4 可信区间与决策概率

可信区间（credible interval）： “给定数据和先验，参数有 95% 的概率落在这个区间内。”——这是管理者能直接理解的表述。

决策概率： “综合防治成功率超过 70% 的概率是多少？”——贝叶斯后验直接给出答案。频率学派不能回答这个问题。

与频率学派置信区间的区别： - 95% CI：“如果重复抽样，95% 的区间会包含真实参数。”（关于程序的陈述） - 95% CrI：“参数有 95% 的概率在这个区间内。”（关于参数的陈述）

20.5 后验预测检查

后验预测检查（posterior predictive check）： 从后验分布中抽取参数，生成模拟数据，比较模拟数据与实际数据的分布。

如果模型合理，模拟数据应该”看起来像”实际数据。如果模拟数据系统性地缺少零值、或者方差太小——模型需要改进。

这是比残差图更直观的模型诊断工具——它直接回答”模型生成的数据是否像真实数据”。

20.6 收敛、有效样本量与链诊断

复杂模型（如层级模型、占域模型）的后验通常没有解析解——需要用 MCMC（马尔可夫链蒙特卡洛）采样。

MCMC 诊断： - R-hat： 多条链之间的收敛诊断。R-hat ≈ 1 表示链已混合，> 1.1 表示未收敛。 - 有效样本量（ESS）： MCMC 样本不是独立的——ESS 告诉你”这些样本等价于多少个独立样本”。ESS 太低说明混合差。 - 链图（trace plot）： 可视化链的轨迹——应该像”毛毛虫”，没有明显的趋势或跳跃。

本章不伪造 MCMC 输出——只解释这些诊断工具的用途和判读标准。

20.7 敏感性和行动边界

贝叶斯分析的最后一个必要步骤：先验敏感性分析。

用不同的合理先验重复分析——结论是否稳定？如果弱信息先验和实质先验给出相似的后验，说明数据足够强。如果后验严重依赖先验，说明数据太弱——需要更多数据或更谨慎的结论。

cat("=== 贝叶斯分析报告 ===\n\n")
cat("问题：综合防治的成功率（受害率 ≤ 30%）是否超过 50%？\n\n")
cat("数据：", n, "个小区中", k, "个成功\n\n")
cat("结论：\n")
cat("- 三种先验下的后验成功率 > 0.5 的概率均较高。\n")
cat("- 结论对先验选择不敏感——数据足够强。\n")
cat("- 可信区间直接回答管理问题。\n\n")
cat("边界：\n")
cat("- 贝叶斯方法不自动解决小样本问题。\n")
cat("- 如果 n=3 且先验很强，后验基本就是先验。\n")
cat("- 先验必须可解释——不能因为'想要显著'而选择乐观先验。\n")

本章小结

  • 贝叶斯更新：先验 × 似然 → 后验。 从已有信念出发，用数据更新，得到新信念。
  • 可信区间直接回答管理者的问题。 “参数有 95% 的概率在这个区间内”——不需要绕弯解释。
  • 先验必须可解释和可检查。 不能因为想要某个结论而选择相应的先验——先验预测检查可以暴露不合理的先验。
  • 后验预测检查：模型生成的数据是否像真实数据？ 比残差图更直观的诊断工具。
  • MCMC 需要诊断。 R-hat、ESS、链图——不收敛的链给出的后验不可信。
  • 先验敏感性分析是必需的。 结论是否依赖先验选择？如果依赖，数据太弱。

练习

  1. 识别层。 以下陈述各属于频率学派还是贝叶斯框架？

    1. “如果零假设为真，观察到当前数据的概率是 0.03。”
    2. “给定数据，效应 > 0 的概率是 0.97。”

  2. 解释层。 为什么”贝叶斯自动解决小样本问题”是错的？

  3. 应用层。 用 Beta-Binomial 模型，计算三种先验下综合防治成功率 > 0.5 的后验概率。结论对先验敏感吗？

  4. 诊断层。 R-hat = 1.5 意味着什么？你应该怎么做？

  5. 综合层。 为黄黄草先导试验设计弱信息先验。说明：你选择了什么分布、什么参数、为什么这是”弱信息”而不是”无信息”。
