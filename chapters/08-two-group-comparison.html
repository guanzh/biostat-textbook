8.1 同样是两组，设计关系可能完全不同

从本章开始，我们进入第三篇：回答常见保护问题。前面七章建立了从设计到数据到可视化的完整链条。现在，你要用这些基础来做正式的统计推断。

两组比较是最简单的推断问题，也是最容易被做错的。不是因为 t 检验的公式难，而是因为”有两组数据”不等于”两组是独立的”。

本章的主项目是森林病虫害防治试验（项目 C）。10 个区组中，每个区组有 4 个处理小区：对照（control）、标准防治（standard）、生物防治（biocontrol）和综合防治（integrated）。本章聚焦于一个预先指定的比较：对照 vs 综合防治。

在开始任何计算之前，先确认设计关系：

  • 处理施加在小区层级（每个区组内随机分配）
  • 每个小区有 20 株树，树木是观测单位，不是实验单位
  • 对照和综合防治的小区在同一个区组内是配对的。它们共享区组的所有条件

这意味着：你不能把 200 株对照树和 200 株综合防治树当作两组独立样本。正确做法是先按小区汇总，然后在区组内做配对比较。

# 读取森林病虫害数据
forest <- read.csv("../data/teaching/forest_pest_trial_long.csv",
                   stringsAsFactors = FALSE)

# 只取对照和综合防治
f2 <- forest[forest$treatment %in% c("control", "integrated"), ]

# 检查设计
table(f2$block_id, f2$treatment)  # 每个区组 × 处理 各 60 行（20 株 × 3 次调查）

观察问题，有人把 200 株对照树和 200 株综合防治树直接做 t 检验，得到 P < 0.001。这个 P 值可信吗？问题出在哪里？

识别单位小区是实验单位
定义差异区组内处理差
匹配设计配对 t 检验
检查失败诊断与敏感性
解释行动效应、区间、边界

学完本章后，你能够：

  1. 根据设计关系（独立/配对/重复测量）选择正确的两组比较方法。
  2. 区分实验单位和观测单位，避免伪重复导致的错误推断。
  3. 使用 Welch t 检验和配对 t 检验，并解释为什么不需要先做方差齐性检验。
  4. 报告效应量（均值差）和置信区间，而不仅仅是 P 值。
  5. 解释置换检验和秩方法的适用边界。它们改变了被检验的对象。

8.2 先定义要估计的差异

“对照和综合防治是否有差异”。这句话不够精确。你必须明确：

  • 目标量（estimand）：你要估计什么？均值差？中位数差？比例差？变化量？
  • 比较尺度：在什么层级上比较？小区均值？树木个体？
  • 预先指定的主要响应：哪个指标是主要判断依据？

本章的主要比较：对照与综合防治在第三次调查时的小区平均虫口密度之差。选择第三次调查是因为防治效果需要时间显现；选择虫口密度是因为它是防治措施最直接的靶标。

次要响应（虫害等级、死亡率、非目标生物数量）将在后续章节处理：本章不基于次要响应反推结论。

# 计算每个小区第三次调查的平均虫口密度
f2_visit3 <- f2[f2$visit == 3, ]
plot_means <- aggregate(
  pest_count ~ block_id + plot_id + treatment,
  data = f2_visit3,
  FUN = mean
)

# 查看数据结构
head(plot_means)
# 每个区组应有 2 行（对照 + 综合防治）
table(plot_means$block_id)

8.3 独立两组：为什么优先考虑 Welch 方法

如果你的两组确实是独立的（例如：完全随机设计中随机分配到两个处理的实验单位），Welch t 检验应该是你的默认选择，不是先做方差齐性检验再决定。

为什么不做方差齐性检验？因为方差齐性检验（如 Levene 检验）本身有功效问题：小样本时检不出方差不齐，大样本时连微不足道的方差差异也能检出。更根本的是，Welch t 检验在方差不齐时仍然有效。它调整了自由度来补偿。所以流程应该是：直接使用 Welch t，不需要方差齐性检验这个中间步骤。

Student t 检验（等方差假设）和 Welch t 检验（不等方差）的区别：

Student t Welch t
方差假设 两组方差相等 不假设方差相等
自由度 n₁ + n₂ − 2 Satterthwaite 近似
方差不齐时 第一类错误率膨胀 仍然稳健
方差齐时 略高的功效 几乎相同的功效

结论：Welch t 在方差齐时几乎不损失功效，在方差不齐时远更可靠。它应该是默认选择。

# 假设我们有两个独立组的虫口密度（仅为演示 Welch t 的语法）

判断：Welch t 检验不要求方差相等。它是两组独立比较的合理默认选择。不需要先做方差齐性检验再决定用哪个 t 检验。

8.4 配对设计：分析的是组内差值

回到主案例。对照和综合防治的小区在同一个区组内是配对的：区组 1 的对照小区和区组 1 的综合防治小区共享区组 1 的所有条件（土壤、坡向、管理历史）。配对 t 检验分析的是区组内的处理差值，而不是两组的原始值。

配对 t 检验的步骤： 1. 在每个区组内计算差值 dᵢ = 综合防治 − 对照 2. 对这 10 个差值做单样本 t 检验：H₀: 差值的均值为 0

这等价于把区组间的变异从误差中扣除：区组效应不再污染处理效应的估计。

# 正确做法：区组内配对比较
# 将数据转为宽格式（每个区组一行，对照和综合防治各一列）
plot_wide <- reshape(
  plot_means[, c("block_id", "treatment", "pest_count")],
  idvar = "block_id",
  timevar = "treatment",
  direction = "wide"
)
names(plot_wide) <- c("block_id", "control", "integrated")

# 计算区组内差值
plot_wide$diff <- plot_wide$integrated - plot_wide$control

# 配对 t 检验
t_test_paired <- t.test(plot_wide$integrated, plot_wide$control,
                         paired = TRUE)
print(t_test_paired)

# 效应量：平均差值
mean_diff <- mean(plot_wide$diff)
cat("平均区组内差（综合防治 - 对照）：", round(mean_diff, 2), "头/株\n")

# 差值的分布
stripchart(plot_wide$diff, method = "jitter", vertical = TRUE,
           pch = 16, col = "#4d8294",
           main = "区组内处理差（综合防治 - 对照）",
           ylab = "虫口密度差（头/株）")
abline(h = 0, lty = 2, col = "gray50")
abline(h = mean_diff, col = "#c86652", lwd = 2)

判断：配对来自设计，不是来自数值相近。区组内的对照和综合防治小区是配对的，因为它们在同一个区组内被随机分配，不是因为它们的虫口密度恰好接近。

8.5 置换与秩方法并非免假设通行证

当数据不满足正态假设时，常有人建议”改用非参数检验”。但非参数检验不是免费的。它们改变了被检验的对象。

置换检验（permutation test）：在零假设下，处理标签是可交换的。通过反复随机打乱标签并计算统计量，得到统计量的零分布。置换检验不假设正态分布，但它假设可交换性：即零假设下，任何分配方案都是等可能的。在配对设计中，可交换性意味着在区组内随机交换处理标签。这与随机区组设计的随机化方案完全一致。

Mann-Whitney U 检验（Wilcoxon 秩和检验）：检验的是两组的中位数是否相等：或者说，从一组随机抽取的观测值大于从另一组随机抽取的观测值的概率是否为 0.5。它不检验均值差。如果你的研究问题确实是”均值是否有差异”，秩检验回答的是一个不同的问题。

# 配对置换检验（区组内交换标签）
set.seed(20260620)
n_perm <- 9999
obs_diff <- mean(plot_wide$diff)
perm_diffs <- replicate(n_perm, {
  # 在每个区组内随机交换处理标签
  swapped <- plot_wide
  for (i in 1:nrow(swapped)) {
    if (runif(1) < 0.5) {
      tmp <- swapped$control[i]
      swapped$control[i] <- swapped$integrated[i]
      swapped$integrated[i] <- tmp
    }
  }
  swapped$diff <- swapped$integrated - swapped$control
  mean(swapped$diff)
})

# 双侧 P 值
p_perm <- mean(abs(perm_diffs) >= abs(obs_diff))
cat("置换检验 P 值:", p_perm, "\n")
cat("配对 t 检验 P 值:", t_test_paired$p.value, "\n")

# 置换分布
hist(perm_diffs, breaks = 50, col = "#e1f5fe",
     main = "置换零分布", xlab = "平均差值")
abline(v = obs_diff, col = "#c86652", lwd = 2)

判断：置换检验不假设正态，但假设可交换性：在配对设计中，这意味着区组内随机交换标签。秩检验改变了被检验的对象。它检验中位数差异，不是均值差异。

8.6 从诊断、效应量和区间解释结果

统计推断的终点不是 P 值。一个完整的分析报告至少包含：

  1. 效应量：综合防治比对照平均减少多少虫口密度？单位是什么？
  2. 置信区间。这个效应的合理范围是多少？
  3. 诊断：模型假设是否合理？是否有异常区组？
  4. 实际意义。这个效应在管理上是否重要？
# 完整分析报告

# 1. 效应量
cat("=== 对照 vs 综合防治：第三次调查虫口密度 ===\n\n")
cat("平均区组内差：", round(mean_diff, 2), "头/株\n")
cat("对照均值：", round(mean(plot_wide$control), 2), "头/株\n")
cat("综合防治均值：", round(mean(plot_wide$integrated), 2), "头/株\n")
cat("相对降低：", round(100 * (1 - mean(plot_wide$integrated) /
                                mean(plot_wide$control)), 1), "%\n\n")

# 2. 置信区间
cat("95% 置信区间：[", round(t_test_paired$conf.int[1], 2), ",",
    round(t_test_paired$conf.int[2], 2), "] 头/株\n\n")

# 3. 诊断：检查差值的分布
cat("差值 Shapiro-Wilk 正态性检验 P =",
    round(shapiro.test(plot_wide$diff)$p.value, 3), "\n")
# 注意：n=10 时正态性检验功效很低：不显著不等于正态

# 检查是否有异常区组
plot_wide$abs_diff <- abs(plot_wide$diff)
plot_wide[order(-plot_wide$abs_diff), c("block_id", "control", "integrated", "diff")]

# 4. 区组间异质性
cat("\n区组间差值的标准差：", round(sd(plot_wide$diff), 2), "\n")
cat("这表明不同区组的防治效果存在差异：\n")
cat("不能假设综合防治在所有区组中效果相同。\n")
黑豹提醒：P 值不是效应大小

P = 0.01 不意味着”效应很大”。它只意味着”如果零假设为真，观察到当前数据的概率很小”。效应大小、置信区间和实际意义必须单独报告。一个很小的效应在大样本下也能 P < 0.001。

8.7 两组证据怎样进入防治决策

统计分析的终点是回答”这些证据对防治决策意味着什么”。对于对照 vs 综合防治的比较：

证据支持：综合防治在第三次调查时平均降低虫口密度约 X 头/株（95% CI [下限, 上限]）。在 10 个区组中，有 Y 个区组显示综合防治优于对照。

证据不支持：综合防治在所有区组中效果一致：区组间差值存在异质性（SD = Z）。不能假设在新区组中效果完全相同。

尚未进入本章分析的因素： - 非目标生物影响（non_target_count）：综合防治是否伤害了天敌？ - 防治成本（cost_cny_plot）：效果是否值得投入？ - 实施完整性（implementation_score）：效果是否依赖于执行质量？ - 这些因素将在第 13、16、17 章中逐步纳入。

反例（树级伪重复）：现在来演示本章最重要的反例：把树木当作独立处理重复。

# ⚠️ 错误做法：树级独立 t 检验（伪重复）
# 这是本章最重要的反例：请仔细阅读为什么它是错的

# 提取对照和综合防治的树级数据（第三次调查）
trees_ctrl <- f2_visit3$pest_count[f2_visit3$treatment == "control"]
trees_int <- f2_visit3$pest_count[f2_visit3$treatment == "integrated"]

# 错误的树级 t 检验
t_wrong <- t.test(trees_int, trees_ctrl)
cat("=== 错误分析：树级独立 t 检验 ===\n")
cat("P 值:", format.pval(t_wrong$p.value, digits = 4), "\n")
cat("均值差:", round(mean(trees_ctrl) - mean(trees_int), 2), "\n")
cat("95% CI: [", round(t_wrong$conf.int[1], 2), ",",
    round(t_wrong$conf.int[2], 2), "]\n")
cat("\n样本量：对照", length(trees_ctrl), "株，综合防治",
    length(trees_int), "株\n")

cat("\n=== 正确分析：区组内配对 t 检验 ===\n")
cat("P 值:", format.pval(t_test_paired$p.value, digits = 4), "\n")
cat("均值差:", round(mean_diff, 2), "\n")
cat("95% CI: [", round(t_test_paired$conf.int[1], 2), ",",
    round(t_test_paired$conf.int[2], 2), "]\n")
cat("\n样本量：", nrow(plot_wide), "个区组\n")

cat("\n=== 为什么树级检验是错的？===\n")
cat("1. 处理施加在小区，不是树木。同一小区的 20 株树共享处理。\n")
cat("2. 树级检验假设 400 个独立观测：实际只有 20 个独立实验单位。\n")
cat("3. 伪重复人为缩小了标准误：区间太窄，P 值太小。\n")
cat("4. 比较两个 P 值：树级检验的 P 值通常远小于配对检验。\n")
cat("   这不是因为'树级检验更灵敏'：是因为它在数学上作弊。\n")

判断：树级 t 检验的 P 值更小，不是因为”方法更灵敏”：是因为伪重复人为缩小了标准误。正确的分析以小区为实验单位，在区组内配对比较。

本章小结

  • “有两组”不足以选择方法。独立、配对和重复测量对应完全不同的分析：方法首先由设计关系决定。
  • Welch t 检验是独立两组比较的合理默认选择。不需要先做方差齐性检验：Welch t 在方差不齐时仍然稳健。
  • 配对来自设计，不是来自数值相近。区组内的处理小区是配对的，因为它们在同一个区组内被随机分配。
  • 置换检验不假设正态，但假设可交换性。秩检验改变了被检验的对象。它检验中位数差异，不是均值差异。
  • P 值不是效应大小。效应量、置信区间和实际意义必须单独报告。
  • 伪重复是两组比较中最致命的错误。把树木当作独立处理重复，等于把 20 个实验单位冒充为 400 个：标准误被人为缩小，P 值被人为夸小。

练习

  1. 识别层。以下场景中，应该用独立 t 检验还是配对 t 检验？为什么？

    1. 10 个区组，每个区组内随机分配对照和处理。
    2. 20 个独立温室，10 个控温 22°C，10 个控温 28°C。
    3. 同一批幼苗，先在光照 A 下测量，再移到光照 B 下测量。

  2. 解释层。为什么树级 t 检验的 P 值比配对 t 检验更小？这是好事还是坏事？

  3. 应用层。用森林病虫害数据，比较标准防治（standard）和生物防治（biocontrol）在第三次调查时的虫口密度。使用正确的配对设计，报告效应量、置信区间和诊断。

  4. 诊断层。配对 t 检验的 95% 置信区间为 [−3.2, −0.4]。请用一句话解释这个区间的含义，并说明它是否支持”综合防治有效”。

  5. 综合层。为黄黄草光照试验设计一个两组比较：比较 100% 光照和 40% 光照在第六个月的开花比例。写出：实验单位、比较结构（独立还是配对？）、目标量、以及为什么不能把植株当作独立重复。
