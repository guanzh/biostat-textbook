# 从样地到结论：保护生物学数据实战

面向野生动物保护、森林保护、植物保护、林学与生物多样性专业的案例驱动统计教材。

基于 R 语言，融入 AI 辅助学习训练。

## 在线阅读

→ https://guanzh.github.io/biostat-textbook

## 技术栈

- [Quarto Book](https://quarto.org/docs/books/)
- R 语言（knitr 引擎）
- GitHub Pages 自动部署

## 教材结构

22 章 + 7 附录，分五篇：

1. 从保护问题进入统计
2. 看懂并整理保护数据
3. 回答常见保护问题
4. 保护专业应用
5. 现代生态统计与继续学习

## 内容成熟度

| 标记 | 含义 |
|------|------|
| `seed` | 仅有标题和方向 |
| `sprout` | 有基本内容但缺案例/数据 |
| `budding` | 内容完整可试教 |
| `mature` | 经过教学验证 |

## 本地构建

```bash
quarto render
```
