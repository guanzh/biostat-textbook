## 一、设计结论

### 1. 章节定位

- 章名（frontmatter title）：**"第 3 章 实验设计：让比较产生可信证据"**。
- **正文不出现 `# H1` 标题**——Quarto 已用 frontmatter title 渲染页面 H1，正文首个标题为 `## 3.1 ...`。这是与第 1、2 章一致的硬约束。
- 不与"调查设计"并列。仅在 3.1 节用一段对比说明观察性研究与操纵实验的结论边界差异；样线、样点、红外相机和抽样努力量留给第 14 章。
- 本章不教 ANOVA、混合模型计算，只建立"设计决定分析"的接口：CRD 指向第 8、9 章，层级和重复测量指向第 18 章。
- 本章终点不是背术语，而是完成一页式实验设计方案，并能指出另一份方案中最致命的设计错误。

### 2. 核心迁移规则

学生离开本章必须能说出并使用这句话：

> **处理在哪一级被独立随机分配，哪一级通常就是实验单位；样本量 n 首先数实验单位，不数测量次数。**

这条规则替代旧版"同一株的叶片 = 技术重复"。同一盆里的多株、同一株的多片叶通常是**子抽样**。技术重复专指同一生物样本在同一测量流程中的重复读数，例如同一 DNA 提取物做三次 qPCR 孔。

### 3. 可观察的学习成果

完成本章后，学生能够：

1. 从研究问题中写出研究对象、处理因素与水平、对照、响应变量和目标总体；
2. 区分实验单位、观测单位、子抽样和技术重复，并据此正确写出样本量；
3. 为具体场景实施简单随机分配或区组内随机分配，并解释随机化防止了什么偏差；
4. 在 CRD 与 RCBD 之间作出有依据的选择，并识别析因、重复测量、嵌套和裂区结构；
5. 从最小有意义效应、自然变异、显著性水平、功效和失访风险说明样本量依据；
6. 写出包含设计、测量、质量控制、伦理与分析接口的一页式实验设计方案。

### 4. 主案例：黄黄草苗圃光照梯度试验

**承接：** 第 2 章中保护站对野外黄黄草指名亚种（_Flos huanghuang_ subsp. _typica_）只能做"年份间观察对比"，无法回答"什么导致开花数下降"。本章把问题升级——

**研究问题：** 林冠郁闭度增加是否抑制黄黄草幼苗的成活与开花？

**为什么是苗圃而不是野外：**
- 黄黄草是濒危物种，野外种群不允许人为施加破坏性处理（伦理边界，3.9 节明确讨论）；
- 保护区合作苗圃中已有近三年由站内野外种子人工繁育的同年龄幼苗，遗传背景可控；
- 苗圃结果将作为未来保护区内**微生境管理建议**的科学依据，而不是野外干预本身。

**初始（错误）方案：**

> 在苗圃东侧露天区选 30 盆幼苗，西侧遮阴棚下选 30 盆，半年后比较两组的开花株比例。

学生首先判断这份方案为什么不能回答光照效应：东西两侧的土壤含水量、风速、苗龄入棚时间、浇水频次都不同，光照与一切都共变。随后改造为目标结构：

- **目标总体**：保护区合作苗圃中由 2024 春季野外种子繁育、目前 18 月龄的黄黄草指名亚种幼苗；
- **实验单位**：苗床上一个独立可遮阴的 1 m² 小区（其上有独立遮阴帘，可施加不同光照处理而互不干扰）；
- **观测单位**：小区内标记的 5 株幼苗；
- **子抽样**：每株上若干叶片的叶绿素读数；
- **技术重复**：同一叶片 SPAD 仪连测三次取均值；
- **处理因素**：光照透过率，4 个水平——全光照（≈100%）、轻度遮阴（70%）、中度遮阴（40%）、重度遮阴（15%）；
- **对照**：全光照水平作为基准对照；中度遮阴对应野外林冠下典型光环境，是预先指定的主要比较；
- **区组**：苗床按南北段划分为若干区组，因为南段更接近水源、北段更接近防风林；
- **分配**：每个区组内将 4 个光照水平随机分配到 4 个小区；
- **主要响应**：6 个月时小区水平的"成活幼苗中开花比例"；
- **次要响应**：株高增量、叶绿素 SPAD 值（明确标注次要）；
- **偏差控制**：基线测量、统一浇水协议、盲法判读开花、随机化种子记录；
- **分析接口**：模型必须保留区组和小区层级，不能把单株幼苗当作独立处理重复。

主案例**不预设固定的小区数**。3.5 节由最小有意义效应和先导数据推导需要的实验单位数，避免一边强调样本量依据，一边凭空规定"每组 10 个"。

## 二、章节内容蓝图（7 节正文 + 小结练习）

正文采用以下 7 个二级标题。每节默认承担约 1 课时（45 分钟），密度大的 3.4 与 3.7 节可拆为两课时；不在教材中印死课时表，由教师据本班节奏自行分配。

### 3.1 从"喷药后变好"到"喷药导致变好"

- 用初始主案例开篇，不立即公布答案；要求学生先写出至少一种非光照解释。
- 明确实验研究主动分配处理，观察性研究只记录自然暴露；随机实验能加强因果解释，但随机化本身不能修复测量偏差、处理污染或严重失访。
- 用"东侧露天 / 西侧遮阴"解释混杂：土壤含水、风速、入棚时间、浇水频次同时变化，无法把开花差异归于光照。
- 一段简短对比，承接第 2 章的"年份对比黄黄草开花数"——指出那是观察性研究，结论必然有边界；本章升级为操纵实验。
- **收束判断：** 实验设计不是数据收集的前奏，而是因果结论能够走多远的边界。

### 3.2 实验单位、观测单位与样本量的真正含义

合并旧版 3.2 + 3.5 节。先讲层级，后讲什么是真正的"重复"——这两件事是同一件事的两面。

- 定义目标总体、实验单位、观测单位、子抽样和技术重复。
- 用主案例的四层结构（区组—小区—幼苗—叶片）演示：处理施加在小区层，所以小区是实验单位；幼苗是观测单位；叶片是子抽样；同一叶片的 SPAD 三次连读才是技术重复。
- **配单位层级 SVG**（本章唯一一张 SVG）：固定四层方框，箭头标明"处理随机分配在哪一层"。
- 加入两个反例：温室里给单株幼苗独立分盆并独立施加处理时，植株可以是实验单位；但若整座温室统一控温，温室才是温度处理的实验单位。
- 纠正"不同对象就一定独立"的误解：相邻小区、同窝动物、同一培养箱中的个体可能共享环境。
- **伪重复诊断三问：** 处理是否独立施加？单位能否在不影响其他单位时接受不同处理？误差项是否与随机化层级一致？
- **收束判断：** 测量单位可以很多，独立处理重复仍可能只有一个。

### 3.3 处理、对照与随机化

合并旧版 3.3 + 3.4 节。对照和随机化是回答同一个问题的两面：怎么让组间差异只反映处理。

- 将处理因素写成可实施的水平：光照透过率必须给出测定方法（如 LAI-2200 林冠分析仪或定标遮阴帘标称值），不能只写"轻度／中度"。
- 区分空白／不处理对照、载体／假处理对照、阳性／标准处理对照——本案例的"全光照"是空白对照，不存在载体对照。
- 说明"控制变量"≠把所有环境条件变成相同；野外或半野外通常做不到，应结合限制条件、区组、随机化和测量协变量。
- 要求学生预先指定**主要响应变量**（开花比例），避免测很多指标后只报告最显著的一个。
- **随机抽样 vs 随机分配：** 前者关系到向谁推广（外部效度），后者关系到组间可比与因果解释（内部效度）。
- 错误做法：按编号奇偶、到达顺序、研究者判断分组都可能与时间趋势或其他因素混杂。
- **R 代码块（演示，不要求学生执行）：** 给出可复现的区组内随机分配代码——

```r
set.seed(20260619)

blocks <- sprintf("B%02d", 1:8)            # 苗床被划分为 8 个区组
treatments <- c("全光照", "70%", "40%", "15%")

allocation <- do.call(
  rbind,
  lapply(blocks, function(block_id) {
    data.frame(
      block = block_id,
      plot  = paste0(block_id, "-P", 1:4),  # 每个区组 4 个小区
      treatment = sample(treatments),       # 区组内随机分配
      stringsAsFactors = FALSE
    )
  })
)

with(allocation, table(block, treatment))   # 每格应等于 1
```

预期输出：32 行分配记录，交叉表每个区组 × 处理格 = 1。代码块下方文字明确告知"如果你已安装 R，可自行复现；否则只看输出即可"。

- 学生必须保存随机种子、分配表和执行日期；不能在看到基线后反复重抽直到"看起来最平衡"。

### 3.4 设计类型谱：从 CRD 到识别复杂结构

合并旧版 3.6 + 3.7 节，但**不平铺六种设计**——而是把它们组织成一条"异质性识别链"：

```
异质性是否已知且强 ─── 否 ──→ CRD（完全随机设计）
                  └─ 是 ──→ RCBD（随机区组设计）
                            │
                            └─ 同时操纵≥2 个因素？──→ 析因
                            └─ 同一单位多次测量？──→ 重复测量
                            └─ 单位天然嵌套于上层？──→ 嵌套
                            └─ 不同因素施加在不同层级？──→ 裂区
```

**本章对各设计的要求层级：**

| 设计 | 本章要求 |
|---|---|
| CRD | **必须能独立设计、写出 n 与分配方法** |
| RCBD | **必须能独立设计、写出区组依据与分配方法** |
| 析因 | 识别结构、知道主效应不能替代交互作用 |
| 重复测量 | 识别结构、知道时间点不是独立重复 |
| 嵌套 | 识别结构、找到处理施加层级 |
| 裂区 | 识别结构、分别指出主区与子区实验单位 |

CRD 与 RCBD 各给一个完整案例：

- **CRD**：同批次苗圃幼苗均一分盆（无明显基线梯度），比较 4 种光照水平。
- **RCBD**：本章主案例——苗床南北段含水量本底差异强，按段分区组。

为两者写一张决策表：已知梯度、随机化范围、误差来源、分析中必须保留的字段、常见失败方式。强调**区组不是越多越好**：与响应无关的区组浪费自由度。

复杂设计部分用一张识别表带过：

| 结构 | 识别信号 | 后续接口 |
|---|---|---|
| 析因 | 同时操纵两个或更多因素 | 第 9、12 章 |
| 重复测量 | 同一实验单位在多个时间点被测量 | 第 18 章 |
| 嵌套 | 样方属于样地、样地属于林分 | 第 18 章 |
| 裂区 | 不同因素在不同层级随机化 | 第 18 章或专业课程 |

**本节末加入第一个黑豹提醒**（标题固定）：

> **"不要数测量次数，要数独立分配次数。"**

### 3.5 样本量不是"每组通常做 10 个"

- 从五个输入解释样本量：**最小有意义效应、单位间变异、显著性水平、目标功效、预计损耗**。
- 两个思想实验：(a) 效应相同但变异更大 → 需要更多实验单位；(b) 测更多叶片只能降低小区均值的测量误差，不能替代更多独立小区。
- 先导试验用于估计变异和检查流程，不应拿极小先导样本的效应量当稳定真值。
- 只要求学生写出**样本量论证链**（"为什么是这个 n"的可追溯叙述），不在尚未学习 P 值和模型前堆积公式；正式数值计算在后续方法章结合对应模型完成。
- 必须报告：实际可用的实验单位数、每单位子抽样数、预期损耗、损耗后的最低有效 n。
- **明确禁止：** 不得给出"本科实验每组至少 3 个""通常每组 10 个"这类无效应量、无变异依据的规则。

### 3.6 测量方案与分析接口

合并旧版 3.9 + 3.10 节。测量协议直接接到数据表结构，再接到分析。

**测量协议必须覆盖：**

- 主要 vs 次要响应、单位、时间点、仪器或分级标准；
- 准确性 vs 精密度的区别——同一开花评级在不同调查员手中不一致就是观察者偏差；
- 调查员培训、盲法判读、仪器校准、重复测量比例、异常值与缺失值记录规则；
- **处理污染、交叉扩散、依从性不足、失访**会破坏最初的处理对比，必须记录而**不能事后删除"不听话"的单位**。

**设计—数据表—分析映射：**

| 设计信息 | 数据表必须保留的字段 | 分析后果 |
|---|---|---|
| CRD、单因素 | 实验单位 ID、处理、响应 | 两组比较或单因素 ANOVA |
| RCBD | 区组 ID、单位 ID、处理、响应 | 必须控制区组 |
| 同一单位重复测量 | 单位 ID、时间、处理、响应 | 观测不独立，不能做普通独立样本检验 |
| 小区内多株幼苗 | 小区 ID、幼苗 ID、处理、响应 | 重复数按小区计算，必要时层级模型 |
| 两因素析因 | 两个因素及水平、响应 | 必须考虑交互作用 |

**收束判断：** 统计软件不能从一张丢失了区组 ID、单位 ID、时间 ID 的"扁平表"中恢复实验设计。

**本节末加入第二个黑豹提醒**（标题固定）：

> **"随机化不能拯救含糊的测量。"**
>
> 80 字内：若开花评级标准、测量时间点或排除规则在中途改变，组间公平比较就失效——再好的随机化也救不回来。

### 3.7 一页式方案、伦理与执行

把"写得别人能执行"的方案变成可交付物。

**12 字段一页式模板（学生必须全部填写）：**

1. 研究问题与可证伪假设；
2. 目标总体、纳入与排除标准；
3. 实验单位、观测单位与层级关系；
4. 处理因素、水平、施加方式与对照；
5. 随机化方法、随机种子、分配记录；
6. 区组或其他限制随机化及其理由；
7. 独立重复数、子抽样数、样本量依据；
8. 主要 / 次要响应、测量时间与质量控制；
9. 缺失、污染、退出与不良事件记录规则；
10. 数据表必须保留的 ID 与分析接口；
11. **伦理、生态安全、人员与时间可行性**——明确说明为何使用苗圃繁育苗而非野外个体；
12. 预先规定的主要比较与结论边界。

**正文为黄黄草主案例提供一份压缩示范**，但样本量字段标注"待先导变异与最小有意义效应填入"，不编造固定 n。

伦理边界部分必须写：

- 濒危物种不能为了实验在野外施加破坏性处理；
- 苗圃结果向野外管理建议外推时，**外部效度有损耗**——学生必须写出"哪些条件不一致使外推有限制"。

## 三、练习与评价闭环

### 1. 章内检查（每节末留 1 题）

每节只放一个短判断或改错题，要求学生先作判断再看解释：

- 3.1：东西两侧苗圃比较开花率为什么不能归因于光照？
- 3.2："30 盆 × 每盆 5 株 = n 150"错在哪？
- 3.3：先看了基线开花数再"调整"分组，为什么破坏随机化？
- 3.4：苗床南北含水量差异强，仍用 CRD 是否一定错？
- 3.5：先导试验里某处理效应巨大，为什么不能直接缩小样本量？

### 2. 章末递进练习（7 题）

1. **识别层：** 在四个场景（含温室、苗圃、林分、培养箱）中标出实验单位、观测单位、子抽样和技术重复。
2. **解释层：** 说明随机化、重复和区组分别控制哪一种不确定性，不能用三个定义互相替代。
3. **应用层：** 为"三种育苗基质对黄黄草发芽率的影响"写出 CRD 设计，包含随机分配方法与主要响应。
4. **分析层：** 批判"一个温室升温、一个温室不升温，每个温室放 100 盆"的结论边界。
5. **综合层：** 使用 12 字段一页式模板完成黄黄草主案例方案。
6. **迁移层：** 将主案例改成"光照 × 浇水频率"两因素析因结构，只画设计结构并指出交互作用问题，不计算模型。
7. **AI 核查：** 让 AI 审查练习 5 的方案；学生必须逐条标注"接受／拒绝／证据不足"，并用实验单位、随机化层级或结论边界说明理由。

### 3. 综合方案评分量规（100 分，8 维度）

| 维度 | 分值 | 满分标准 | 一票否决式错误 |
|---|---:|---|---|
| 问题与因果对比 | 10 | 问题、假设、主要比较一致 | 处理与结论不对应 |
| 单位与层级 | 20 | 正确区分实验单位、观测单位、子抽样 | 把子抽样当独立处理重复 |
| 处理与对照 | 10 | 水平可执行、对照能排除关键替代解释 | 没有可解释的对照 |
| 随机化与区组 | 15 | 分配过程可复现、区组理由成立 | 按地点或研究者偏好系统分组 |
| 重复与样本量 | 15 | n 数的是实验单位、依据链完整 | 用测量次数冒充独立 n |
| 测量与偏差控制 | 15 | 主要响应、时间、盲法、质量控制明确 | 结果出来后才选主要指标 |
| 数据与分析接口 | 10 | ID、层级、时间和主要比较均可从数据恢复 | 数据表无法恢复随机化结构 |
| 伦理与可行性 | 5 | 风险、污染、执行约束有处理方案 | 方案实际无法执行或风险不可接受 |

出现"一票否决式错误"时，该维度不超过一半分；不把整份作业判零，因为学生仍需看见其他环节是否正确。
